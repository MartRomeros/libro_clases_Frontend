# Spec: Centralizar estilos css

Se debe realizar una mejora en cuanto el css de esta aplicacion por lo que se realizara lo siguiente:
todos los componentes de angular material no se deben modificar su estilo.
Solamente en casos muy especificos se podra modificar el estilo de un componente de angular material.

utilizar los colores de angular material que estan establecidos en este archivo `src/material-theme.scss`.
los estilos que son reutilizables deben quedar en este archivo `styles.css`.
